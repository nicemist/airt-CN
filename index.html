3 条命令即可开始

每个 Lab 都通过 Docker 在本地运行。克隆仓库，选择 Lab，开始你的攻击演练。

terminal 
# 下载并解压 Labs
curl -LO airt-labs.zip
unzip airt-labs.zip -d airt-labs

# 启动任意 Lab（例如 Lab 01 - 基础）
cd lab01-foundations

# 访问 Lab 交互界面

# 使用 garak 运行漏洞扫描

# 启动 PyRIT Orchestrator
python -m pyrit.orchestrator --config config.yaml
About AIRT

社区驱动的 AI 安全教育

AI Red Team Academy 是一个免费开源的教育资源，旨在普及 AI 安全知识。我们相信理解攻击技术是构建强大 AI 防御的关键。
